# Product Catalog: Component Hierarchy

| Level | Component | Description |
| :--- | :--- | :--- |
| **Atoms** | Button, Input, Typography, Badge | Basic building blocks like a "Buy" button or search field. |
| **Molecules** | ProductCard, SearchBar, FilterCheckbox | Combinations of atoms, like a card representing a listing]. |
| **Organisms** | ProductGrid, NavigationHeader, FilterSidebar | Distinct sections that manage layout and repeated rendering. |
| **Templates** | CatalogLayout | The skeletal structure for the sidebar and grid. |
| **Pages** | MainCatalogPage | The final view populated with real database data. |
